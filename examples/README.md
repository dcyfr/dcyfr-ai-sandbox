# @dcyfr/ai-sandbox Examples

This folder contains lightweight sandbox examples for rapid experimentation.

## Prerequisites

- Node.js `>=24.13.0`
- From `dcyfr-ai-sandbox/`: `npm install`

## Example Index

| File                | Demonstrates                                    | Run Command               |
| ------------------- | ----------------------------------------------- | ------------------------- |
| `basic-usage.js`    | minimal harness setup and loader initialization | `npm run example:basic`   |
| `plugin-system.js`  | plugin workflow outline                         | `npm run example:plugins` |
| `agent-patterns.js` | agent lifecycle pattern overview                | `npm run example:agents`  |

## Validation Commands

- Syntax check all examples: `npm run examples:compile`
- Run all examples: `npm run examples:check`
- Full validation: `npm run examples:test`

## Expected Output

Each example prints a scenario header and ends with a human-readable completion message.
