## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/dcyfr-ai-sandbox.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature-name`

## Development

This is an experimentation sandbox, so contributions can include:

- **New Examples** - Demonstrate framework features
- **Test Cases** - Validate framework behavior
- **Utilities** - Helper functions for common tasks
- **Documentation** - Improve guides and explanations

## Testing

Run tests before submitting:
```bash
npm run test:run
npm run typecheck
npm run lint
```

## Submitting Changes

1. Commit your changes with clear messages
2. Push to your fork
3. Create a Pull Request
4. Describe what you've added/changed

## Code Style

- Follow existing patterns
- Use TypeScript where applicable
- Keep examples simple and focused
- Document complex logic

Open an issue or discussion on GitHub!
